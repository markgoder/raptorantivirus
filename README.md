# raptorantivirus
Raptonvntivirus is a unique antivirus that bypasses Trojans. It's designed to improve performance and communicate with the task manager remotely, for example, if a virus blocks the task manager via the registry.
RaptorAntivirus — это легкая и эффективная утилита для мониторинга системы Windows и управления активными процессами в реальном времени. Разработана для быстрого выявления подозрительной активности и завершения зависших или нежелательных задач.
🚀 Основные возможности
Мониторинг ресурсов: Отслеживание нагрузки на CPU, RAM и GPU.
Диспетчер процессов: Удобный список активных программ с возможностью завершения процесса одним кликом.
Системные инструменты: Быстрый доступ к «Диспетчеру задач» и системным настройкам.
Защита: Встроенный модуль для предотвращения блокировки системных функций вирусами.

📥 Как установить и запустить
Перейдите в раздел Releases.
Скачайте последнюю версию raptorantivirus.exe.
Запустите файл от имени администратора (нажмите правой кнопкой мыши на файл -> «Запуск от имени администратора»).
Примечание: Поскольку программа выполняет системные вызовы, антивирусы могут выдать ложное срабатывание. Программа безопасна, так как её исходный код открыт.
